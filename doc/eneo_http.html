error/4

error(ResCode::pos_integer(), Errcode::atom(), Error::binary(), Req::cowboy_req:req()) -> cowboy_req:req()

Sends an error back to the client. Note that Errcode is specified as lowercase atom and should correspond to matrix common error code (see @url(https://spec.matrix.org/latest/client-server-api/#common-error-codes). For backward compatibility, Errcode can be passed as binary.

parse_body/1

parse_body(Req0::cowboy:req()) -> {ok, map(), cowboy:req()}

Reads the entire http body of a cowboy request object and returns the body as an erlang map. Throws {invalid_json, Req} if the http body is invalid json.

read_entire_body(Req0::cowboy:req()) -> {binary(), cowboy:req()}

reply/3

reply(ResponseCode::pos_integer(), Data::map() | [any()], Req::cowboy_req:req()) -> cowboy_req:req()

Sends a reply back to the client. Takes in a Map as Data, so that an arbitrary format can be returned, e.g. plain html, json, xml. Currently, only json is supported
