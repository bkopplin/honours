init/2

init(Req::cowboy_req:req(), Opts::[atom()]) -> {ok, cowboy:req(), [atom()]}

Serves as an entry point for the cowboy router. Takes in a request object and a list of options where the first element is the action to perform. Actions correspond to matrix endpoints. See the matrix documentation and the http_action() type specification for more information on which actions match which matrix endpoint. For instance, the Action atom "messages" corresponds to the matrix endpoint at /rooms/{room_id}/messages.
